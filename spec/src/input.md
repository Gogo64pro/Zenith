# Input format

## Source Encoding
Source encoding is **implementation specific**

UTF-8 is recommended for implementations

In the https://github.com/Gogo64pro/Zenith compiler it is utf-8 recommended, any bytes/invalid utf accepted

## Line endings
No normalization

CRLF and LF are treated as valid line endings

The CR in CRLF doesn't count as a column

CR only line endings not accepted
