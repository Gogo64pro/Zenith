# Introduction

This book is the primary reference of Zenith.

## This language is a work in progress
Zenith is a work in progress, the language has no implementation yet and syntax is subject to change

## This is a language *specification*
This book is a language specification, it describes the syntax and semantics of the language, but it is not a tutorial

### This spec is inspired by the [Rust language specification](https://doc.rust-lang.org/stable/reference/introduction.html)
